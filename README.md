# MotorPHGUI2 - Payroll System with Java Swing

MotorPHGUI2 is a Java-based payroll system developed with NetBeans and Java Swing. It simulates a functional HR and payroll platform for a fictional company, **MotorPH**, enabling HR personnel to manage employee records, track attendance, and calculate salaries with a graphical user interface.

## Features

-  **Graphical User Interface (GUI)** using Java Swing
-  **Employee Information Management** (View, Add, Update employee data)
-  **Attendance Logging System** (Daily in/out time records)
-  **Automated Salary Computation**
  - Regular hours
  - Overtime
  - Late/Undertime deductions
  - Government deductions (SSS, PhilHealth, Pag-IBIG)
-  **CSV Report Export** (Employee data and payroll summary)
-  **User Login System**
-  **Real-time Payroll Breakdown Display**
-  **Modular and Organized Java Class Design**

##  Technologies Used

- Java (JDK 8+)
- Java Swing (GUI)
- NetBeans IDE
- CSV file handling for data persistence (no database used)
